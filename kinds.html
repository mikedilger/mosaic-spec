Record Kinds

PAGE STATUS: Early Draft

Kinds are 2-byte unsigned integers in little-endian format.

This page is a Registry of kinds, their name, and the standard that defines them. Refer to their definitions in the standards that define them.

Kind Name Standard Server Used
0x1 Key Schedule Mosaic Core Key Schedule Outbox
0x2 Profile Mosaic Core Profile Outbox
0x3 Microblog Root Mosaic Social Media Microblog Root Outbox
0x4 Reply Comment Mosaic Social Media Reply Comment Outbox & Parent Author Inbox
0x5 Blog Post Mosaic Social Media Blog Post Outbox
0x6 Chat Message Mosaic Social Media Chat Message Chat Server
