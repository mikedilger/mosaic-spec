Record Kinds

PAGE STATUS: Early Draft

Kinds are 4-byte unsigned integers in little-endian format.

This page is a Registry of kinds, their name, and the standard that defines them. Refer to their definitions in the standards that define them.

Kind Name Standard
0x1 Key Schedule Mosiac Core Key Schedule
0x2 Profile Mosaic Core Profile
0x3 Microblog Root Mosaic Social Media Microblog Root
0x4 Reply Comment Mosaic Social Media Reply Comment
0x5 Blog Post Mosaic Social Media Blog Post
0x6 Chat Message Mosaic Social Media Chat Message
